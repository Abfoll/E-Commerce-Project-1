# Ecommerce Backend API

Backend API for ecommerce platform built with Node.js, Express, PostgreSQL, and Sequelize.

## Features
- User authentication & authorization
- Product management
- Order processing
- Category management
- File uploads
- Admin dashboard

## Deployment
- **Backend**: Render.com
- **Database**: Neon.tech (PostgreSQL)
- **Frontend**: Vercel
